Описание работы с файлами НК



Данная страница НК предназначена для загрузки и выгрузки различной информации (файлов) в НК
Документация - выгрузка файла документации по НК (manual.pdf) если он размещен вместе файлами web морды. Обычно этот файл отсутствует.
Текущее состояние ТН - выгрузка текстового файла в котором описано текущее состояние ТН - настройки и значения датчиков. Состояние считывается в момент запроса.
Сохранение настроек ТН в текстовом виде - выгрузка настроек НК в текстовый файл, не все настройки сохраняются, т.к. при появлении новых настроек необходимо дописывать код для сохранения в текстовом виде, а порой про это забывается. Полезная настройка для изучения текущих настроек или переноса настроек между разными версиями или разными ТН (когда требуется ручное задание настроек).
Прочитать всю EEPROM - выгрузка содержимого i2c EEPROM в бинарный файл. Возможно это требуется для клонирования НК
Сохранение настроек ТН в бинарном виде - выгрузка настроек НК в бинарном виде. Сохраняются все настройки, профили и расписание.
Восстановление настроек ТН из бинарного файла - загрузка (восстановление) сохраненных бинарных настроек в НК. При не совпадении форматов (размера данных), восстанавливаются только те настройки которые есть в версии НК в который загружается файл

Web интерфейс контроллера
Выбор файлов для записи их в контроллер - появляется окно выбора файлов web морды. Надо понимать что web морда может храниться на чипе spi flash и/или на SD карте.
При хранении на чипе spi flash есть особенность. Формат хранения не предполагает удаление и добавление отдельных файлов (мне было лень это реализовывать). По этому при записи происходит стирание ВСЕХ файлов а потом происходит запись выбранных файлов. Из этого следует, что выбирать надо ВСЕ файлы web морды для записи. Процесс занимает определенной время и в этот момент не надо уходить/обновлять текущую страницу, по его завершению откроется сообщение.
При хранении на SD карте файлы с одинаковыми именами заменяются а новые добавляются.
Выбор источника загрузки (sd карта или spi флеш) происходит на странице Сервис->Настройки->"Расположение файлов веб-сервера на SPI Flash (иначе на SD карте)".

Работа ТН
Журнал работы ТН - выгрузка журнала (текстовый формат) работы ТН. Актуальна для изучения проблем работы НК и при возникновении ошибок.
Графики работы ТН из ОЗУ - выгрузка графиков (csv формат) в текстовом формате. Актуальна при необходимости загрузки графиков в электронные таблицы или базы данных.
Статистика работы ТН (файлы по годам) - выгрузка статистики работы (csv формат) в текстовом формате. Статистика это усредненные результаты за каждый день. При выгрузке надо выбрать файл (хранится по годам).Если файлов нет на карте то список для выбора пустой.
История работы ТН (файлы по годам) - еслив опциях включена данная функция НК сохраняет информацию каждые 5 минут в файл. Эту информацию можно выгрузить в текстовом формате.При выгрузке надо выбрать файл (хранится по годам).Если файлов нет на карте то список для выбора пустой.
