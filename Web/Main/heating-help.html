Описание параметров отопления/охлаждения


Алгоритм работы - выбор режима работы для ТН с частотным управлением компрессором (инверторный ТН). Возможно варианты:
1. HYSTERESIS - режим старт-стопа с выбором целевой температуры;
2. PID - Гистерезис для температуры в доме + ПИД регулирование (стабилизация) температуры подачи;
3. HYBRID - пока не реализован.
Для ТН старт-стоп будет работать только первый вариант.
Целевая температура (что является целью) - Для алгоритма HYSTERESIS можно выбирать что будет целевой температурой. для алгоритма PID - желательно установить целью температуру дома
Целевая температура в доме - значение целевой температуры в доме
Целевая температура обратки - значение целевой температуры обратки (для алгоритма HYSTERESIS и выбора цели - обратка)

Использование ночного тарифа - блок параметров описывающих логику использования ночного тарифа (при много тарифном учете). Основная идея - дополнительно (избыточно) нагреть дом по ночному (более дешевому) тарифу. Для правильной настройки необходимо понимать время ночного тарифа вашего электро счетчика и выставлять в настройках НК его границах(часы НК так-же должны быть выставлены правильно).
Добавка к целевой температуре по часам - на сколько градусов увеличивается целевая температура при работе по ночному тарифу. Для отключения использования "ночного тарифа" установите значение в ноль.
Начальный час добавки к целевой температуре - начало работы (увеличения целевой температуры) по ночному тарифу
Конечный час добавки к целевой температуре - окончание работы (увеличения целевой температуры) по ночному тарифу

Погодозависимость - блок параметров описывающих логику погодозависимости. Погодозависимая автоматика (ПА) - это комплекс программных и аппаратных средств для обеспечения автоматического изменения некоторых параметров системы отопления в соответствии с колебаниями температуры окружающего воздуха. Например:На улице холодает, температура теплоносителя растет, на улице теплеет температура теплоносителя снижается. Задача такого алгоритма работы – поддержание температуры воздуха в помещении на заданном уровне. В НК реализовано два метода учета погодозависимости:
1. Изменение целевой температуры (дома или обратки) для работы по гистерезису (возможно применение для алгоритмов HYSTERESIS и PID) в зависимости от уличной температуры.
2. Изменение целевой температуры подачи при работы ПИДа на подачу (только для алгоритма PID), на улице холодает - температура подачи растет, теплеет - температура подачи падает.
Базовая температура - формула: цель += (база - улица) * К (первый метод), для отключения первого метода установите 0 в это поле.
Максимальное изменение цели на C° - ограничение на изменение цели (первый метод)
Коэффициент погодозависимости цели - коэффициент пересчета (первый метод)
Использование погодозависимости для ПИД - включение второго метода погодозависимости для алгоритма PID
Коэффициент погодозависимости для ПИД - коэффициент пересчета для второго метода

ПИД - блок параметров ПИД регулятора при выборе алгоритма PID (регулирование подачи)
Целевая температура подачи - цель работы ПИД регулятора, что является целью при стабилизации температуры подачи.
Постоянная интегрирования времени - как часто происходит расчет и корректировка частоты компрессора ПИДом. Обычно это 30-120 секунд.
Пропорциональная составляющая - Пропорциональная составляющая вырабатывает выходной сигнал, противодействующий отклонению регулируемой величины от заданного значения, наблюдаемому в данный момент времени. Он тем больше, чем больше это отклонение. Если входной сигнал равен заданному значению, то выходной равен нулю.
Однако при использовании только пропорционального регулятора значение регулируемой величины никогда не стабилизируется на заданном значении. Существует так называемая статическая ошибка, которая равна такому отклонению регулируемой величины, которое обеспечивает выходной сигнал, стабилизирующий выходную величину именно на этом значении. Например, в регуляторе температуры выходной сигнал (мощность нагревателя) постепенно уменьшается при приближении температуры к заданной, и система стабилизируется при мощности, равной тепловым потерям. Температура не может достичь заданного значения, так как в этом случае мощность нагревателя станет равна нулю, и он начнёт остывать.
Чем больше коэффициент пропорциональности между входным и выходным сигналом (коэффициент усиления), тем меньше статическая ошибка, однако при слишком большом коэффициенте усиления при наличии задержек (запаздывания) в системе могут начаться автоколебания, а при дальнейшем увеличении коэффициента система может потерять устойчивость.
Интегральная составляющая - Интегрирующая составляющая пропорциональна интегралу по времени от отклонения регулируемой величины. Её используют для устранения статической ошибки. Она позволяет регулятору со временем учесть статическую ошибку.
Если система не испытывает внешних возмущений, то через некоторое время регулируемая величина стабилизируется на заданном значении, сигнал пропорциональной составляющей будет равен нулю, а выходной сигнал будет полностью обеспечиваться интегрирующей составляющей. Тем не менее, интегрирующая составляющая также может приводить к автоколебаниям при неправильном выборе её коэффициента.
Дифференциальная составляющая - Дифференцирующая составляющая пропорциональна темпу изменения отклонения регулируемой величины и предназначена для противодействия отклонениям от целевого значения, которые прогнозируются в будущем. Отклонения могут быть вызваны внешними возмущениями или запаздыванием воздействия регулятора на систему.

Опции
Теплый пол - если есть отдельный насос (имя RPUMPFL) на теплый пол то включение в работу этого насоса.
Использовать солнечный коллектор (TSUN>TEVAOUTG+Δ°) - использование солнечного коллектора для отопления, если есть солнечный коллектор.

Защиты при работе теплового насоса - блок параметров обеспечивающих аварийную защиту ТН. Выход за пределы этих параметров приводит к аварийному останову ТН и выдачи ошибки.
Минимальная/максимальная температура подачи - придельная температура подачи для охлаждения/отопления.
Максимальная/минимальная температура обратки - придельная температура обратки для охлаждения/отопления.
Максимальная разность температур на конденсаторе - придельная разность температур между подачей и обраткой для охлаждения/отопления.

Ежесуточное включение реле - дополнительный функционал, который позволяет включать одно из реле НК (выходы) по расписанию.
