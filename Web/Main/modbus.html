Адрес устройства на шине RS485:
Частотник: 1, Счетчик: 2. Нумерация параметров с 1.

Чтение, запись1. Адреса 4...: Адреса 3...:

Устройство Параметр, адрес2 Значение Статус
Примечание:
Необходимо понимать, что разные устройства передают по разному многобайтные регистры (какой байт идет первым), по этому для чтения float необходимо пробовать оба варианта Float и I(Float).
Например: прочитать напряжение счетчика SDM120 - установить адрес устройства на шине 2, добавить строку вывода, нажав кнопку I(Float) и ввести адрес в строке вывода 1 (в документации указано 0х0000), прочитать регистр нажав кнопку '>'. Для чтения частоты из счетчика надо указать адрес 71 (в документации указано 0х0046), все остальное аналогично.

1. Для корректного отображения информации из регистра необходимо правильно указать формат хранимого значения в регистре (см. документацию на устройство).
2. Нумерация ведется с еденицы (обычно необходимо добавить 1 к адресу), значения вводятся в ДЕСЯТИЧНОМ формате
